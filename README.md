# Higher Lower Game 📱
A follower count guessing game built with Python.

## What it does

* Picks two random celebrities/accounts from a dataset of 50
* Displays their name, description, and country
* Player guesses who has more followers
* Keeps score and chains correct answers into the next round
* Ends the game immediately on a wrong guess

## What I learned

* Working with external data files and dictionaries
* Random selection with duplicate prevention
* Chaining game state between rounds
* Boolean logic for win condition checking

## How to run

1. Clone the repository
2. Make sure `game_data.py` is in the same directory
3. Run `python main.py`
4. Type 'A' or 'B' to guess who has more followers
